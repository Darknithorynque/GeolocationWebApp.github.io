# Geolocation-Web-App
This work is a geolocation web app creating by using React library. It provides to show your current location is in the which continent in the world
If you want to try how it's work, you have to allow cookie or you see "Geolocation is denied" message or kind of something on browser.
If you want to see how look the app in other continents you can manage your geolocation by following these steps; right click > inspect > click three dot in the top right > select "sensors" > manage the geolocation
